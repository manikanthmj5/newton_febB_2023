Image tag 
//To display image inside the browser we using image tag 
//Image we don't have closing tag 
//It is inline tag
//inline element if you don't have space it will go to next line
example:
<img>
Attributes:
src - source - url of the image 
alt  - alternate - suppose image is not present we it will give the content
width - values in number - 300
height - values in number  - 400
id 
class 
Image extension 
//we can use .png or .jpeg or .jpg 
//Avoid negative value for width and height 
example:
<img src="car.png">
Problems if image is not coming 
1.Check image is present in current folder 
2.check name of the image and extension is matching with your src path
