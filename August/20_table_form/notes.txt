Table:
//It consists of row and column
//To create row we need to use tr tag 
//To create column header we need to use th tag
//To create column cell we need to use td tag
example:
<table>
<tr>
<th>Name</th>
</tr>
</table>
Attributes:
1.width 
2.bgcolor -->background color 
3.align
4.borderColor ---> color for the border
5.border ----->size of the border 
Division in Table
1.thead - all headers related column we can put inside the thead 
2.tbody - main content we can put inside this
3.tfoot - last column related to ending of tag 
Purpose of Division in Table 
1.To maintain the code readability 
2.To apply some css styles we can make use of this division.
