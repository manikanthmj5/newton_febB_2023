Display Property:
//html already predefined the propery of display 
//by using css display property we can able to change them from block to inline or inline to block.
1.block - it will not allow another element in same line.
example:
display:block;
2.inline - everything will continue in same line.
example:
display:inline;
3.inline-block - it will work similar to inline property.
//Main difference between inline and inline-block 
//inline will not support width and height property 
//inline-block will support width and height property
example:
display:inline-block;
4.display property for table:
//without table tag we can able to create tabular format 
display:table ---> create table 
display:table-row ---> similar tr tag 
display:table-cell --> similar to td tag
